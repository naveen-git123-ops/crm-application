# Task Management - Complete User Guide

## 🎯 How to Open a Task

### Method 1: Click on Task Card (Recommended)
1. **Locate the Task**: Find the task you want to open in the Kanban board (Pending, In Progress, Completed, or Overdue columns)
2. **Click the Task Card**: Click anywhere on the task card
3. **View Details**: A side panel will slide in from the right showing all task details
4. **Close Panel**: Click the X button at the top right of the panel or click outside to close

### What You'll See When a Task Opens:
- Task ID and Title at the top
- Tab navigation with 4 options:
  - **Details** - Task information and settings
  - **Comments** - Add and view comments
  - **Time** - Log time and view time logs
  - **Attachments** - Upload and download files

---

## ✏️ How to Edit a Task

### Edit Task Details:
1. **Open the Task**: Click on any task card
2. **Go to Details Tab**: It should be open by default
3. **Click "Edit Task"**: Look for the blue "Edit Task" button with a pencil icon
4. **Modify Fields**: You can now edit:
   - **Title**: Change the task name
   - **Description**: Update task details
   - **Priority**: Select Low, Medium, or High
   - **Status**: Change to Pending, In Progress, Completed, or Overdue
   - **Due Date**: Pick a new date from the calendar
   - **Completion Percentage**: Use the slider to set 0-100%
5. **Save Changes**: Click the green "Save" button
6. **Confirmation**: You'll see a success message

### What Each Field Does:
- **Title**: The main task name (visible on the card)
- **Description**: Detailed information about the task
- **Priority**: Affects the card color on the board
- **Status**: Determines which column the task appears in
- **Due Date**: When the task should be completed
  - If due date passes and task isn't completed, it auto-moves to "Overdue"
- **Completion %**: Visual indicator of progress (0-100%)
  - Shows as a progress bar on the task card

## 💬 How to Add Comments

### Add a Comment:
1. **Open the Task**: Click on the task card
2. **Go to Comments Tab**: Click "Comments" in the tab navigation
3. **Type Your Comment**: In the text field at the top, type your message
4. **Send Comment**: Press Enter or click the send button (📤 icon)
5. **View Comments**: All comments appear below with:
   - Author name
   - Date posted
   - Comment text

### Comment Features:
- **Real-time updates**: Comments appear immediately after posting
- **Author tracking**: See who made each comment
- **Thread organized**: Comments stay with the task

---

## 📎 How to Add Attachments

### Upload a File:
1. **Open the Task**: Click on the task card
2. **Go to Attachments Tab**: Click "Attachments" in the tab navigation
3. **Upload File**: 
   - Click on the dashed box that says "Click to upload or drag files"
   - Select a file from your computer, OR
   - Drag and drop a file into the box
4. **Confirmation**: You'll see a success message
5. **View File**: The uploaded file appears in the list with:
   - File name
   - Who uploaded it
   - Upload date
   - Download link (📥 icon)

### Working with Attachments:
- **Download**: Click the download icon next to any attachment to save it
- **Multiple Files**: You can upload multiple files to one task
- **File Types**: Supports all common file types (PDF, images, documents, etc.)

---

## ⏱️ How to Log Time

### Log Time on a Task:
1. **Open the Task**: Click on the task card
2. **Go to Time Tab**: Click "Time" in the tab navigation
3. **Fill in Time Log Form**:
   - **Date**: Select the date when you worked on the task
   - **Time Spent (minutes)**: Enter how many minutes you worked (e.g., "60" for 1 hour)
   - **Description**: Optional - explain what you worked on
4. **Submit**: Click "Log Time" button
5. **View Logs**: All time logs appear below the form with:
   - Who logged the time
   - Date of work
   - Minutes spent
   - Work description (if provided)

### Time Tracking Tips:
- Log time regularly to track progress
- Use descriptions to document what was accomplished
- Total time helps managers see productivity

## 📊 Admin/Manager Dashboard (Admins & Managers Only)

### Access Dashboard:
1. **Find Dashboard Tab**: In the header of the Tasks page, next to "Board" tab
2. **Click "Dashboard"**: Switch to dashboard view

### Dashboard Features:

#### Summary Cards:
- **Total Tasks**: All tasks in the system
- **Pending**: Tasks waiting to be started
- **In Progress**: Tasks being worked on
- **Completed**: Finished tasks
- **Overdue**: Tasks past their due date

#### Employee Task Table:
Shows for each employee:
- **Name**: Employee name
- **Total**: Number of tasks assigned
- **Pending**: Count of pending tasks
- **In Progress**: Count of in-progress tasks
- **Completed**: Count of finished tasks
- **Overdue**: Count of overdue tasks
- **Avg Completion %**: Average completion percentage across all tasks
  - Visual progress bar shows completion status

#### Filter by Employee:
1. **Use Dropdown**: In the header, select an employee name
2. **View Their Tasks**: Board updates to show only that employee's tasks
3. **See Individual Tasks**: A task list appears below the table showing all their tasks with completion percentages
4. **Reset Filter**: Select "All Employees" to see all tasks again

## 🎨 Understanding Task Colors & Status

### Status Columns:
- **Pending** (Gray): Task not yet started
- **In Progress** (Blue): Currently being worked on
- **Completed** (Green): Task is finished
- **Overdue** (Red): Task is past due date and not completed

### Priority Colors (Card Background):
- **Low** (Light Blue): Can wait, not urgent
- **Medium** (Light Yellow): Should be done soon
- **High** (Light Red): Urgent, needs attention

### Auto-Status Changes:
- If a task's due date passes and it's not yet "Completed" or "Overdue", it **automatically moves to the Overdue column**
- This happens when you refresh or access the board

---

## 🔄 Moving Tasks Between Columns

### Drag and Drop:
1. **Click and Hold**: Click and hold on a task card
2. **Drag to New Column**: Drag the card to a different status column
3. **Drop**: Release the mouse to drop it in the new column
4. **Auto-Save**: Status updates immediately

### Direct Status Change:
1. **Open Task**: Click on task card
2. **Go to Details Tab**
3. **Click Edit Task**
4. **Change Status Dropdown**: Select new status
5. **Save**: Click Save button

## ❓ Troubleshooting

### Task Won't Open?
- Make sure you're clicking on the task card (not dragging)
- Try clicking the title or task ID
- Look for the side panel sliding in from the right

### Changes Not Saving?
- Check for error messages (red toast notifications)
- Verify you clicked the "Save" button in Edit mode
- Wait for green success message

### Can't See Comments/Attachments?
- Make sure you're on the correct tab
- Try refreshing the page
- Check if other users added them (page auto-refreshes)

### File Upload Failed?
- Check file size (should be reasonable)
- Try a different file type
- Check internet connection
- Look for error message in notification

## 🔐 Permissions

### All Users (Employees & Admins):
- View assigned tasks
- Add comments
- Upload attachments
- Log time
- Update task completion %
- Change task status

### Admins & Managers Only:
- View Dashboard
- Filter tasks by employee
- See all tasks (not just assigned)
- Create new tasks
- Edit any task details

### Regular Employees:
- See only their own assigned tasks
- Cannot see Dashboard
- Cannot filter by employee
- Cannot access Admin features
